# Contributing

This is an open source, community-driven project.
If you'd like to contribute, please start by reading the following documents:

* [Code of conduct][0]
* [MIT License][1]

[0]: ./CODE_OF_CONDUCT.md
[1]: https://choosealicense.com/licenses/mit/

## Reporting a bug

- Use the Github Issues page to report a bug. 
- Please be the more descriptive possible when reporting an issue:
   - PHP version (if not used in docker stack) 
   - OS and specs
   - Use case
   - What was expected ?
   - What happened ?
- Choose correct labels for your issue (frontend, backend, devops ...)

## Submitting a Pull Request

Any contribution is welcome :beers: !

- Fork this repository
- Create a branch with correct prefix: `bugfix-` `feat-` `hotfix-` etc..
- Submit your Pull Request on this repository
